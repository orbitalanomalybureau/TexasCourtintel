# Texas Court Intel MVP -> Professional Grade Roadmap

## Phase A (Now - 2 weeks)
- Professional UI/UX pass
- Search/filter/export stable
- Per-field source attribution
- Last-verified/freshness badges
- Private notes (firm-only) with local storage fallback

## Phase B (2-4 weeks)
- Backend API (FastAPI/Node)
- PostgreSQL storage
- Real user auth (email + RBAC)
- Admin panel with save/edit history
- Source register database

## Phase C (4-8 weeks)
- Multi-user collaboration notes
- Alerting (rule/court/staff updates)
- Court profile completeness scoring
- Audit trail + basic compliance dashboard

## Phase D (8-12 weeks)
- Billing/subscriptions
- Team management + seat limits
- Reporting + trend analytics
- Pilot onboarding playbook

## Minimum Professional Requirements
- Attribution on each fact
- Freshness dates visible
- Dispute/correction workflow
- Activity logs
- Role-based access controls
- Terms, privacy policy, acceptable use
